# AI Search Pro (浏览器扩展)

AI Search Pro 是一款强大的浏览器扩展插件。当你在传统的搜索引擎（如百度、Google、Bing）中进行搜索时，它会在网页侧边栏或悬浮小窗中，同步唤起多个主流 AI 大模型（如豆包、ChatGPT、通义千问、DeepSeek、Kimi 等），并自动将你的搜索词发送给它们，让你在看传统搜索结果的同时，获得 AI 的智能解答。

## ✨ 核心特性

- **🚀 极速静默加载**：在搜索引擎首页输入关键词并回车的瞬间，插件会静默拦截并存储搜索词。当跳转到结果页时，小窗**零延迟同步秒出**，AI 已经开始为你生成答案。
- **🔄 无缝延续对话**：如果你在搜索结果页重新输入新词搜索，小窗内的 AI 会在**当前对话中继续回答**你的新问题，绝不会粗暴地刷新页面或开启新对话，保留你的上下文。
- **🎨 智能悬浮与侧边栏双模式**：
  - **悬浮小窗模式**：精美通透的弥散阴影，支持拖拽移动。自动识别网页布局，智能向下挤压（700px）网页内容，不会遮挡“相关搜索”或“百度热搜”。
  - **侧边栏模式**：一键切换为原生的侧边栏，支持拖拽左侧边缘调整宽度。
- **⚙️ 高度可定制的设置面板**：
  - 支持开启/关闭你不需要的 AI 平台。
  - 支持拖拽排序，将你最常用的 AI 助手排在前面。
  - 所有配置自动本地缓存，刷新页面也能瞬间恢复你习惯的布局。

## 🛠️ 安装方法

1. 下载或克隆本仓库到本地。
2. 打开 Chrome/Edge 浏览器，在地址栏输入 `chrome://extensions/`（或 `edge://extensions/`）进入扩展程序管理页面。
3. 在页面右上角开启 **“开发者模式”**。
4. 点击左上角的 **“加载已解压的扩展程序”**，选择本仓库所在的文件夹（即包含 `manifest.json` 的目录）。
5. 安装成功！建议将插件固定在浏览器的工具栏上，方便随时点击唤出/关闭。

## 💡 使用指南

1. **正常搜索**：打开百度、Google 等搜索引擎，像平时一样搜索，AI 助手会自动在右侧弹出。
2. **切换平台**：在小窗顶部的 Tab 栏，点击不同的图标即可无缝切换各个 AI 的回答。
3. **调整顺序与开关**：点击小窗右上角的“⚙️ 设置”图标，通过拖拽调整平台顺序，或使用开关关闭不需要的平台。点击“保存并应用”后即刻生效，且不会打断当前对话。
4. **拖拽与布局**：按住小窗顶部的拖拽图标可任意移动位置；移回原位时，网页会自动腾出空间。
5. **模式切换**：点击小窗右上角的侧边栏图标，可将悬浮窗固定为右侧边栏；在侧边栏模式下，鼠标悬停在左侧边缘可左右拖拽调整宽度。

## 👨‍💻 开发者信息

- **架构**：采用纯原生 JavaScript/CSS 编写，无第三方框架依赖，极致轻量。
- **核心机制**：通过 `sessionStorage` 与 `postMessage` 实现跨页面极速传参与 iframe 内部通信。采用 `document.documentElement` 挂载方案，免疫各大搜索引擎的 SPA 局部刷新导致的 DOM 销毁。
